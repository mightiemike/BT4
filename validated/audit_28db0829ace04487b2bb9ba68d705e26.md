### Title
Migration Halts on Unbonding Validator: `transferDelegationFromPosition` Rejects Non-Bonded Validators During v8 Force-Exit — (`x/tieredrewards/keeper/transfer_delegation.go`)

---

### Summary

`transferDelegationFromPosition` contains an unconditional `IsBonded()` guard that returns `ErrValidatorNotBonded` for any validator not in the `Bonded` state. During the v8 migration, `exitVestedAccountsPositions` calls `ForceFullExitWithDelegation` for every vesting-owned position. If any such position is delegated to a validator that began unbonding before the upgrade block, the guard fires, the error propagates unhandled all the way through `Migrate1to2`, and the upgrade handler fails — halting the chain.

---

### Finding Description

**Root cause — `transfer_delegation.go` lines 132–134:**

```go
if !validator.IsBonded() {
    return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, types.ErrValidatorNotBonded
}
``` [1](#0-0) 

This guard exists in `transferDelegationFromPosition` — the *exit* direction — even though the analogous guard in `transferDelegationToPosition` (line 53–55) is the one that makes semantic sense (blocking new positions on non-bonded validators). [2](#0-1) 

For the exit direction the operation is: `Unbond(posDelAddr, valAddr, shares)` then `Delegate(owner, amount, validator.GetStatus(), validator, false)`. Both calls are valid on an unbonding validator in the Cosmos SDK — `Unbond` works regardless of validator status, and `Delegate` with `subtractAccount=false` accepts any status. The `IsBonded()` check is therefore a spurious guard that blocks a legitimate migration path.

**Error propagation chain:**

1. `transferDelegationFromPosition` → `ErrValidatorNotBonded`
2. `ForceFullExitWithDelegation` line 62–64 wraps and returns the error [3](#0-2) 
3. `exitVestedAccountsPositions` line 99–101 wraps and returns the error [4](#0-3) 
4. `Migrate` line 37–39 wraps and returns the error [5](#0-4) 
5. `Migrate1to2` returns the error to the Cosmos SDK upgrade handler [6](#0-5) 

In Cosmos SDK, an upgrade handler that returns an error causes the node to panic and halt. There is no recovery path in the migration code — no `continue`-on-error, no skip logic, no fallback.

---

### Impact Explanation

**Primary impact — chain halt at the upgrade block.** Any single vesting-owned position delegated to a validator whose status is `Unbonding` or `Unbonded` at the upgrade block causes `Migrate1to2` to return an error, which panics the upgrade handler and halts the chain. All nodes halt simultaneously at the same block height.

**Secondary impact — position permanently locked (if chain somehow continues).** The position is never deleted, the delegation remains on the position's delegator address, and the owner has no user-facing transaction to recover it post-migration (the `ForceFullExitWithDelegation` path is the only migration-time exit for vesting accounts). [7](#0-6) 

---

### Likelihood Explanation

Validators begin unbonding routinely — due to governance jailing, self-undelegation, or operator decisions. The unbonding period on Cronos POS Chain is 28 days. Any validator that began unbonding within 28 days before the upgrade block will still be in `Unbonding` status at upgrade time. If even one vesting account holds a position on such a validator, the chain halts. An adversary who controls a validator and holds a vesting-account position can deliberately trigger this by initiating validator unbonding shortly before the upgrade.

---

### Recommendation

Remove the `IsBonded()` guard from `transferDelegationFromPosition`. The guard is semantically correct only for the *to-position* direction. For the *from-position* direction, `Unbond` and `Delegate` both accept non-bonded validators. Alternatively, if the guard must remain for normal user-facing exits, add a bypass flag or a separate migration-only code path that calls `stakingKeeper.Unbond` + `stakingKeeper.Delegate` directly without the bonded check. [8](#0-7) 

---

### Proof of Concept

```
1. Create a vesting account (e.g. ContinuousVestingAccount).
2. Call LockTier to create a delegated position on validator V.
3. Begin unbonding validator V (e.g. via MsgEditValidator / jailing or
   direct staking keeper call in an integration test).
4. Assert validator V status == Unbonding.
5. Run Migrator.Migrate1to2(ctx).
6. Assert: Migrate1to2 returns an error containing "ErrValidatorNotBonded".
7. Assert: the position still exists in the Positions store.
8. Assert: the delegation on the position's delegator address still exists.
```

The test can be written as a standard Go integration test using the existing `keeper_helpers_test.go` scaffolding. No governance, no privileged keys, and no external dependencies are required — only a vesting account, a position, and a validator status transition before calling `Migrate1to2`. func (k Keeper) transferDelegationFromPosition(ctx context.Context, pos types.PositionState, valAddr sdk.ValAddress, amount math.Int) (math.LegacyDec, math.LegacyDec, math.Int, error) {
	owner, err := sdk.AccAddressFromBech32(pos.Owner)
	if err != nil {
		return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, errorsmod.Wrap(sdkerrors.ErrInvalidAddress, "invalid owner address")
	}

	posDelAddr, err := sdk.AccAddressFromBech32(pos.DelegatorAddress)
	if err != nil {
		return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, errorsmod.Wrap(sdkerrors.ErrInvalidAddress, "invalid delegator address")
	}

	// Defensive
	if !pos.IsDelegated() {
		return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, errorsmod.Wrapf(types.ErrPositionNotDelegated, "position %d is not delegated", pos.Id)
	}

	// Defensive
	isRedelegating, err := k.isRedelegating(ctx, pos.DelegatorAddress)
	if err != nil {
		return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, err
	}
	if isRedelegating {
		return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, errorsmod.Wrapf(types.ErrActiveRedelegation, "position %d has an active redelegation", pos.Id)
	}

	validator, err := k.stakingKeeper.GetValidator(ctx, valAddr)
	if errors.Is(err, stakingtypes.ErrNoValidatorFound) {
		return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, types.ErrTransferDelegationDestNotFound
	} else if err != nil {
		return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, err
	}

	if !validator.IsBonded() {
		return math.LegacyDec{}, math.LegacyDec{}, math.Int{}, types.ErrValidatorNotBonded
	}

func (k Keeper) ForceFullExitWithDelegation(ctx context.Context, posID uint64) error {
	logger := k.logger(ctx)
	logger.Info("force-exit: begin", "position_id", posID)

	posState, err := k.getPositionState(ctx, posID)
	if err != nil {
		return fmt.Errorf("get position %d: %w", posID, err)
	}
	if !posState.IsDelegated() {
		logger.Error("force-exit: position is not delegated; cannot force full exit",
			"position_id", posID,
			"owner", posState.Owner,
		)
		return nil
	}
	logger.Info("force-exit: position state loaded",
		"position_id", posID,
		"owner", posState.Owner,
		"tier_id", posState.TierId,
		"validator", posState.Delegation.ValidatorAddress,
		"shares", posState.Delegation.Shares.String(),
	)

	posState, baseRewards, bonusRewards, err := k.claimRewards(ctx, posState)
	if err != nil {
		return fmt.Errorf("claim rewards for position %d: %w", posID, err)
	}
	logger.Info("force-exit: claimed rewards",
		"position_id", posID,
		"base_rewards", baseRewards.String(),
		"bonus_rewards", bonusRewards.String(),
	)

	valAddr, err := sdk.ValAddressFromBech32(posState.Delegation.ValidatorAddress)
	if err != nil {
		return fmt.Errorf("parse validator address for position %d: %w", posID, err)
	}

	positionAmount, err := k.reconcileAmountFromShares(ctx, valAddr, posState.Delegation.Shares)
	if err != nil {
		return fmt.Errorf("reconcile amount for position %d: %w", posID, err)
	}
	logger.Info("force-exit: reconciled position amount",
		"position_id", posID,
		"amount", positionAmount.String(),
		"validator", valAddr.String(),
	)

	if _, _, _, err := k.transferDelegationFromPosition(ctx, posState, valAddr, positionAmount); err != nil {
		return fmt.Errorf("transfer delegation back to owner for position %d: %w", posID, err)
	}

func exitVestedAccountsPositions(
	ctx context.Context,
	positions collections.Map[uint64, types.Position],
	ak AccountKeeper,
	pk PositionForceExiter,
) error {
	sdkCtx := sdk.UnwrapSDKContext(ctx)

	var toExit []uint64
	if err := positions.Walk(ctx, nil, func(posID uint64, pos types.Position) (bool, error) {
		ownerAddr, err := sdk.AccAddressFromBech32(pos.Owner)
		if err != nil {
			return false, fmt.Errorf("parse owner address for position %d: %w", posID, err)
		}
		acc := ak.GetAccount(ctx, ownerAddr)
		if acc == nil {
			return false, fmt.Errorf("owner account not found for position %d: %s", posID, ownerAddr.String())
		}
		if _, ok := acc.(sdkvesting.VestingAccount); ok {
			toExit = append(toExit, posID)
		}
		return false, nil
	}); err != nil {
		return fmt.Errorf("walk positions: %w", err)
	}

	for _, posID := range toExit {
		sdkCtx.Logger().Info("v8 migration: force-exit vesting-owned position", "position_id", posID)
		if err := pk.ForceFullExitWithDelegation(ctx, posID); err != nil {
			return fmt.Errorf("force-exit position %d: %w", posID, err)
		}
	}

	sdkCtx.Logger().Info("v8 migration: vesting-owned positions exited", "count", len(toExit))
	return nil
