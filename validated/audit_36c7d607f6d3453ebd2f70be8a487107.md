### Title
Division by Zero in `PerpEngine.socializeSubaccount` When `openInterest` Is Zero Blocks Liquidation — (File: `core/contracts/PerpEngine.sol`)

---

### Summary

`PerpEngine.socializeSubaccount` divides by `state.openInterest` with no zero-guard. When a perp product has no open positions (`openInterest == 0`) but a subaccount still carries a negative `vQuoteBalance` that exceeds the insurance fund, the division reverts unconditionally, permanently blocking liquidation of that subaccount and preventing bad-debt socialization.

---

### Finding Description

In `PerpEngine.socializeSubaccount`, after insurance is applied, the residual bad debt is spread across all open-interest holders:

```solidity
if (balance.vQuoteBalance < 0) {
    // socialize across all other participants
    int128 fundingPerShare = -balance.vQuoteBalance.div(
        state.openInterest          // ← division by zero if openInterest == 0
    );
