### Title
Permanent Loss of Accrued Bonus Rewards When Bonus Pool Is Empty During `BeforeRedelegationSlashed` — (`x/tieredrewards/keeper/slash.go`, `claim_rewards.go`)

### Summary

`slashRedelegationPosition` calls `processEventsAndClaimBonus`, which deletes validator events and advances position checkpoints (`LastEventSeq`, `LastBonusAccrual`) **before** checking pool sufficiency. When the pool is empty, `ErrInsufficientBonusPool` is returned with the position already mutated. `slashRedelegationPosition` swallows this error and then calls `setPositionWithState`, permanently persisting the advanced checkpoints. All bonus accrued up to the slash is irrecoverably lost.

### Finding Description

`processEventsAndClaimBonus` processes events in a loop that, for each event entry:

1. Calls `pos.UpdateLastEventSeq(entry.Seq)` — advancing the in-memory checkpoint past the event.
2. Calls `k.decrementEventRefCount(ctx, valAddr, entry.Seq)` — deleting the event from `ValidatorEvents` storage when its `ReferenceCount` reaches zero. [1](#0-0) 

After the loop, `applyBonusAccrualCheckpoint` advances `pos.LastBonusAccrual` to the current block time: [2](#0-1) 

Only **after** all these mutations does the function check pool balance: [3](#0-2) 

If the pool is empty, `ErrInsufficientBonusPool` is returned — but the `pos` pointer already has `LastEventSeq` advanced and `LastBonusAccrual` updated, and the events are already deleted from storage.

In `slashRedelegationPosition`, this error is explicitly swallowed: [4](#0-3) 

Then `setPositionWithState` is called unconditionally, persisting the corrupted position: [5](#0-4) 

`setPositionWithState` writes the position to `k.Positions`: [6](#0-5) 

On any subsequent call to `processEventsAndClaimBonus` for this position, `getValidatorEventsSince` starts from the now-advanced `LastEventSeq` — finding no events — and `computeSegmentBonus` uses the now-advanced `LastBonusAccrual` as `segmentStart`, computing zero bonus for the entire pre-slash period. [7](#0-6) 

### Impact Explanation

Every position owner whose redelegation is slashed while `RewardsPoolName` has insufficient balance permanently loses all bonus rewards accrued since their last claim. The accrued bonus is non-recoverable: the events are deleted, the checkpoints are advanced, and no replay is possible. This is a direct, permanent loss of user funds (bonus rewards) with no recourse.

### Likelihood Explanation

The `RewardsPoolName` module account balance is finite and can reach zero through normal operation (all bonuses claimed, no new funding). A redelegation slash is a normal chain event triggered by validator double-signing or downtime. The two conditions — pool empty + redelegation slash — can coincide naturally without any attacker action. An adversary who can observe pool balance could also time a slash (e.g., by triggering downtime on a validator they control) to maximize victim losses.

### Recommendation

Move the `sufficientBonusPoolBalance` check **before** the event-processing loop in `processEventsAndClaimBonus`. If the pool is insufficient, return early without mutating `pos` or decrementing any event reference counts. Alternatively, use a two-phase approach: compute the total bonus first (read-only pass), check pool sufficiency, then — only if sufficient — decrement ref counts and advance checkpoints. In `slashRedelegationPosition`, if `ErrInsufficientBonusPool` is swallowed, the position's `LastEventSeq` and `LastBonusAccrual` must **not** be persisted (i.e., do not call `setPositionWithState` with the mutated `pos`).

### Proof of Concept

1. Create a position with a redelegation to validator V.
2. Advance time so bonus accrues across several validator events.
3. Drain RewardsPoolName to zero (e.g., by claiming all other positions' bonuses).
4. Trigger BeforeRedelegationSlashed for the position's unbondingId.
5. Assert: ValidatorEvents for V is now empty (all ref counts decremented to zero).

7. Assert: position.LastBonusAccrual == block time at slash (advanced).

func (k Keeper) processEventsAndClaimBonus(ctx context.Context, pos *types.PositionState) (sdk.Coins, error) {
	// Rewards should have been claimed before undelegation
	if !pos.IsDelegated() {
		return sdk.NewCoins(), nil
	}

	valAddr, err := sdk.ValAddressFromBech32(pos.Delegation.ValidatorAddress)
	if err != nil {
		return nil, err
	}

	events, err := k.getValidatorEventsSince(ctx, valAddr, pos.LastEventSeq)
	if err != nil {
		return nil, err
	}

	sdkCtx := sdk.UnwrapSDKContext(ctx)
	blockTime := sdkCtx.BlockTime()

	totalBonus := math.ZeroInt()
	// Use the persisted bonded state from the last replay, not a hardcoded default.
	// This prevents overpaying bonus for unbonded gaps between claims.
	bonded := pos.LastKnownBonded
	segmentStart := pos.LastBonusAccrual

	tier, err := k.getTier(ctx, pos.TierId)
	if err != nil {
		return nil, err
	}

	for _, entry := range events {
		evt := entry.Event

		if bonded {
			// Compute bonus for the bonded segment [segmentStart, eventTime]
			// using the snapshot rate at the event.
			bonus := k.computeSegmentBonus(*pos, tier, segmentStart, evt.Timestamp, evt.TokensPerShare)
			totalBonus = totalBonus.Add(bonus)
		}

		// Update bonded state based on event type.
		switch evt.EventType {
		case types.ValidatorEventType_VALIDATOR_EVENT_TYPE_UNBOND:
			bonded = false
		case types.ValidatorEventType_VALIDATOR_EVENT_TYPE_BOND:
			bonded = true
		case types.ValidatorEventType_VALIDATOR_EVENT_TYPE_SLASH:
			// Slash doesn't change bonded state.
		}

		segmentStart = evt.Timestamp
		pos.UpdateLastEventSeq(entry.Seq)

		// Decrement reference count.
		if err := k.decrementEventRefCount(ctx, valAddr, entry.Seq); err != nil {
			return nil, err
		}
	}

	val, err := k.stakingKeeper.GetValidator(ctx, valAddr)
	if err != nil {
		return nil, err
	}
	// Defensive: validator bond status check
	if bonded && val.IsBonded() {
		currentRate, err := k.getTokensPerShare(ctx, valAddr)
		if err != nil {
			return nil, err
		}
		bonus := k.computeSegmentBonus(*pos, tier, segmentStart, blockTime, currentRate)
		totalBonus = totalBonus.Add(bonus)
	}

	applyBonusAccrualCheckpoint(&pos.Position, blockTime)
	// Persist the bonded state so the next replay starts correctly.
	pos.UpdateLastKnownBonded(bonded)

	if totalBonus.IsZero() {
		return sdk.NewCoins(), nil
	}

	bondDenom, err := k.stakingKeeper.BondDenom(ctx)
	if err != nil {
		return nil, err
	}

	bonusCoins := sdk.NewCoins(sdk.NewCoin(bondDenom, totalBonus))

	if err := k.sufficientBonusPoolBalance(ctx, bonusCoins); err != nil {
		return nil, err
	}

func (k Keeper) slashRedelegationPosition(ctx context.Context, unbondingId uint64, sharesToUnbond math.LegacyDec) error {
	positionId, err := k.getRedelegationMapping(ctx, unbondingId)
	if errors.Is(err, collections.ErrNotFound) {
		return nil
	}
	if err != nil {
		return err
	}

	pos, err := k.getPositionState(ctx, positionId)
	if errors.Is(err, types.ErrPositionNotFound) {
		k.logger(ctx).Error("position not found during redelegation slash",
			"position_id", positionId,
			"unbonding_id", unbondingId,
			"error", err.Error(),
		)
		return nil
	}
	if err != nil {
		return err
	}

	if !pos.IsDelegated() {
		// Defensive
		k.logger(ctx).Error("delegation missing during BeforeRedelegationSlashed",
			"position_id", positionId,
			"unbonding_id", unbondingId,
			"shares_to_unbond", sharesToUnbond.String(),
		)
		return nil
	}

	dstValStr := pos.Delegation.ValidatorAddress

	// Settle bonus against PRE-slash shares.
	if _, err := k.processEventsAndClaimBonus(ctx, &pos); err != nil {
		// Deliberately forgo bonus rewards if pool is insufficient to prevent chain halt.
		if errors.Is(err, types.ErrInsufficientBonusPool) {
			k.logger(ctx).Error("insufficient bonus pool during redelegation slash",
				"position_id", pos.Id,
				"error", err.Error(),
			)
		} else {
			return err
		}
	}

	fullSlash := sharesToUnbond.GTE(pos.Delegation.Shares)

	if fullSlash {
		pos.Delegation = nil
		pos.ClearBonusCheckpoints()
		return k.setPositionWithState(ctx, pos, &ValidatorTransition{PreviousAddress: dstValStr})
	}
	// In-memory only: the persisted Position carries no share count, and the
	// live delegation will reflect the post-Unbond shares on the next read.
	// Update the local copy so any follow-up logic in this call sees consistent state.
	pos.Delegation.Shares = pos.Delegation.Shares.Sub(sharesToUnbond)
	return k.setPositionWithState(ctx, pos, nil)

func (k Keeper) decrementEventRefCount(ctx context.Context, valAddr sdk.ValAddress, seq uint64) error {
	key := collections.Join(valAddr, seq)
	event, err := k.ValidatorEvents.Get(ctx, key)
	if errors.Is(err, collections.ErrNotFound) {
		return nil // already cleaned up
	}
	if err != nil {
		return err
	}

	if event.ReferenceCount <= 1 {
		return k.ValidatorEvents.Remove(ctx, key)
	}

	event.ReferenceCount--
	return k.ValidatorEvents.Set(ctx, key, event)
}
