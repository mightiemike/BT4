### Title
`BurnNFT` Requires Caller to Be Both NFT Owner and Denom Creator Simultaneously, Permanently Locking NFTs Minted to Non-Creator Recipients — (`x/nft/keeper/keeper.go`)

### Summary

`keeper.BurnNFT` enforces two independent ownership checks on the **same address**: `IsOwner` (NFT owner) and `IsDenomCreator` (denom creator). When an NFT is minted to a recipient who is not the denom creator, neither party can ever satisfy both checks, making the NFT permanently unburnable.

### Finding Description

`keeper.BurnNFT` performs these two sequential checks:

```go
// keeper.go lines 146-154
nft, err := k.IsOwner(ctx, denomID, tokenID, owner)
if err != nil {
    return err
}

_, err = k.IsDenomCreator(ctx, denomID, owner)
if err != nil {
    return err
}
``` [1](#0-0) 

Both checks must pass for the **same** `owner` argument. `IsOwner` verifies the caller holds the NFT token, while `IsDenomCreator` verifies the caller created the denom — these are independent roles that can be held by different addresses. [2](#0-1) [3](#0-2) 

`MintNFT` explicitly supports minting to a separate `recipient` address (distinct from `sender`/creator):

```go

if err := m.Keeper.MintNFT(ctx, msg.DenomId, msg.Id,
    msg.Name, msg.URI, msg.Data,
    sender,     // must be denom creator
    recipient,  // NFT owner — can be any address
); err != nil {
``` [4](#0-3) 

### Impact Explanation

Call sequence that produces a permanently unburnable NFT:

1. `IssueDenom(creator)` — creator becomes denom creator
2. `MintNFT(sender=creator, recipient=alice)` — alice becomes NFT owner
3. `BurnNFT(denomID, tokenID, creator)` → **fails**: `IsOwner` rejects creator (alice is owner)
4. `BurnNFT(denomID, tokenID, alice)` → **fails**: `IsDenomCreator` rejects alice (creator is denom creator)

No address can satisfy both checks simultaneously. The NFT is permanently locked from burning. The same deadlock occurs if alice later transfers the NFT to bob — bob also cannot satisfy `IsDenomCreator`.

### Likelihood Explanation

This is triggered by any standard `MsgMintNFT` with `recipient != sender`, which is an explicitly supported and documented production flow. No special privileges, governance, or key compromise required. Any user can reach this state with two ordinary transactions.

### Recommendation

`BurnNFT` should allow burning if the caller satisfies **either** condition (is the NFT owner **or** is the denom creator), not both simultaneously. A corrected guard:

```go
_, ownerErr := k.IsOwner(ctx, denomID, tokenID, owner)
_, creatorErr := k.IsDenomCreator(ctx, denomID, owner)
if ownerErr != nil && creatorErr != nil {
    return sdkerrors.Wrapf(types.ErrUnauthorized, "...")
}
```

Integration test outline (standard Go keeper test, no mocks needed):

func (k Keeper) BurnNFT(ctx sdk.Context, denomID, tokenID string, owner sdk.AccAddress) error {
	if !k.HasDenomID(ctx, denomID) {
		return sdkerrors.Wrapf(types.ErrInvalidDenom, "denom ID %s not exists", denomID)
	}

	nft, err := k.IsOwner(ctx, denomID, tokenID, owner)
	if err != nil {
		return err
	}

	_, err = k.IsDenomCreator(ctx, denomID, owner)
	if err != nil {
		return err
	}

	k.deleteNFT(ctx, denomID, nft)
	k.deleteOwner(ctx, denomID, tokenID, owner)
	k.decreaseSupply(ctx, denomID)

	return nil
}

func (k Keeper) IsOwner(ctx sdk.Context, denomID, tokenID string, owner sdk.AccAddress) (types.BaseNFT, error) {
	nft, err := k.GetNFT(ctx, denomID, tokenID)
	if err != nil {
		return types.BaseNFT{}, err
	}

	if !owner.Equals(nft.GetOwner()) {
		return types.BaseNFT{}, sdkerrors.Wrapf(types.ErrUnauthorized, "%s is not the owner of %s/%s", owner.String(), denomID, tokenID)
	}

	return nft.(types.BaseNFT), nil
}

func (k Keeper) IsDenomCreator(ctx sdk.Context, denomID string, address sdk.AccAddress) (types.Denom, error) {
	denom, err := k.GetDenom(ctx, denomID)
	if err != nil {
		return types.Denom{}, err
	}

	creator, err := sdk.AccAddressFromBech32(denom.Creator)
	if err != nil {
		panic(err)
	}

	if !creator.Equals(address) {
		return types.Denom{}, sdkerrors.Wrapf(types.ErrUnauthorized, "%s is not the creator of %s", address, denomID)
	}

	return denom, nil

func (m msgServer) MintNFT(goCtx context.Context, msg *types.MsgMintNFT) (*types.MsgMintNFTResponse, error) {
	recipient, err := sdk.AccAddressFromBech32(msg.Recipient)
	if err != nil {
		return nil, err
	}

	sender, err := sdk.AccAddressFromBech32(msg.Sender)
	if err != nil {
		return nil, err
	}

	ctx := sdk.UnwrapSDKContext(goCtx)
	if err := m.Keeper.MintNFT(ctx, msg.DenomId, msg.Id,
		msg.Name,
		msg.URI,
		msg.Data,
		sender,
		recipient,
	); err != nil {
		return nil, err
	}

	ctx.EventManager().EmitEvents(sdk.Events{
		sdk.NewEvent(
			types.EventTypeMintNFT,
			sdk.NewAttribute(types.AttributeKeyTokenID, msg.Id),
			sdk.NewAttribute(types.AttributeKeyDenomID, msg.DenomId),
			sdk.NewAttribute(types.AttributeKeyTokenURI, msg.URI),
			sdk.NewAttribute(types.AttributeKeyRecipient, msg.Recipient),
		),
		sdk.NewEvent(
			sdk.EventTypeMessage,
			sdk.NewAttribute(sdk.AttributeKeyModule, types.AttributeValueCategory),
			sdk.NewAttribute(sdk.AttributeKeySender, msg.Sender),
		),
	})
