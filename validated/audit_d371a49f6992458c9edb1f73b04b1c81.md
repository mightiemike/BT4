### Title
Malicious IBC Counterparty Can Drain Escrowed Native NFTs via Crafted ClassId in Sink Direction — (`x/nft-transfer/keeper/packet.go`)

### Summary

`processReceivedPacket` contains no guard in the sink (back-to-origin) branch that verifies the incoming `data.ClassId` corresponds to a voucher that was legitimately minted on the counterparty chain. A malicious counterparty operator can craft a packet whose `data.ClassId` is prefixed with their own source port/channel, forcing `IsAwayFromOrigin` to return `false`, stripping the prefix to recover a bare native class ID, and then calling `TransferOwner` to move a native NFT out of the escrow address to an arbitrary receiver — without ever having held the corresponding voucher.

---

### Finding Description

**Entrypoint:** IBC `OnRecvPacket` → `processReceivedPacket`

The sink-direction branch in `processReceivedPacket` is: [1](#0-0) 

```go
} else {
    unprefixedClassID := types.RemoveClassPrefix(packet.GetSourcePort(),
        packet.GetSourceChannel(), data.ClassId)

    voucherClassID := types.ParseClassTrace(unprefixedClassID).IBCClassID()
    for _, tokenID := range data.TokenIds {
        if err := k.nftKeeper.TransferOwner(ctx,
            voucherClassID, tokenID, escrowAddress, receiver); err != nil {
            return err
        }
    }
}
```

The escrow address used here is: [2](#0-1) 

```go
escrowAddress := types.GetEscrowAddress(packet.GetDestPort(), packet.GetDestChannel())
```

This is the escrow address for **our chain's** port/channel (the destination side of the arriving packet).

**`IsAwayFromOrigin` logic:** [3](#0-2) 

```go
func IsAwayFromOrigin(sourcePort, sourceChannel, fullClassPath string) bool {
    prefixClassID := GetClassPrefix(sourcePort, sourceChannel)
    return !strings.HasPrefix(fullClassPath, prefixClassID)
}
```

If the attacker's source port is `nft` and source channel is `channel-0`, and they set `data.ClassId = "nft/channel-0/nftClass"`, then:

- `GetClassPrefix("nft", "channel-0")` → `"nft/channel-0/"`
- `strings.HasPrefix("nft/channel-0/nftClass", "nft/channel-0/")` → `true`
- `IsAwayFromOrigin(...)` → `false` (sink direction triggered)

**`RemoveClassPrefix` then strips the prefix:** [4](#0-3) 

```go
func RemoveClassPrefix(portID, channelID, classID string) string {
    classPrefix := GetClassPrefix(portID, channelID)
    return classID[len(classPrefix):]
}
```

`"nft/channel-0/nftClass"[len("nft/channel-0/"):]` → `"nftClass"`

**`ParseClassTrace("nftClass").IBCClassID()`:** [5](#0-4) 

Since `"nftClass"` has no `/`, `classSplit[0] == rawClassID` → `ClassTrace{Path: "", BaseClassId: "nftClass"}`. [6](#0-5) 

`IBCClassID()` with empty `Path` returns `"nftClass"` directly (no `ibc/` hash prefix).

So `voucherClassID = "nftClass"` — a bare native class ID.

**`TransferOwner` is then called with the escrow address as source:** [7](#0-6) 

If the escrow address at `GetEscrowAddress("nft", "channel-X")` holds `"nftClass"/"tokenID"` (because a legitimate user previously escrowed it), `TransferOwner` succeeds and the NFT is moved to the attacker's receiver.

**Missing guard:** The away-from-origin branch correctly checks `k.HasClassTrace` before proceeding: [8](#0-7) 

```go
if !k.HasClassTrace(ctx, classTrace.Hash()) {
    k.SetClassTrace(ctx, classTrace)
}
```

The sink branch has **no analogous check** — it does not verify that the `unprefixedClassID` corresponds to a class trace that was previously registered when the NFT was sent out via this channel.

**`ValidateBasic` provides no protection:** [9](#0-8) 

It only checks that `ClassId` is non-blank, `TokenIds` is non-empty, lengths match, and addresses are valid bech32. It does not validate the semantic meaning of `ClassId`.

---

### Impact Explanation

A malicious counterparty chain operator can drain any native NFT that was legitimately escrowed at the escrow address for the channel they are connected to. The attacker does not need to hold the corresponding IBC voucher. The NFT is transferred to an arbitrary receiver address the attacker specifies. This is a direct, permanent theft of user-owned NFTs.

---

### Likelihood Explanation

Any IBC channel connected to a malicious or compromised counterparty chain is exploitable. The attacker only needs to know which native NFT class IDs and token IDs are currently held in the escrow address (which is public on-chain state). The crafted packet requires no special privileges beyond being the counterparty chain operator.

---

### Recommendation

In the sink-direction branch of `processReceivedPacket`, add a check that the `unprefixedClassID` (after prefix removal) corresponds to a class trace that was previously registered on this chain via the away-from-origin path. Specifically, verify that `k.HasClassTrace(ctx, classTrace.Hash())` returns `true` before calling `TransferOwner`, and return an error if it does not. This mirrors the existing guard in the away-from-origin branch and ensures that unescrow can only occur for NFTs that were legitimately sent out via this channel.

---

### Proof of Concept

**Setup:**
1. Chain A (victim) has native NFT class `"nftClass"`, token `"token1"`, owned by `userA`.
2. Chain A has IBC channel `(nft, channel-X)` connected to attacker's chain `(nft, channel-0)`.
3. `userA` legitimately sends `"nftClass"/"token1"` via IBC → NFT is escrowed at `GetEscrowAddress("nft", "channel-X")`.

**Attack:**
4. Attacker crafts a packet from their chain with:
   - `SourcePort = "nft"`, `SourceChannel = "channel-0"`
   - `DestPort = "nft"`, `DestChannel = "channel-X"`
   - `data.ClassId = "nft/channel-0/nftClass"`
   - `data.TokenIds = ["token1"]`
   - `data.Receiver = attacker_address`
5. IBC core delivers the packet to Chain A's `OnRecvPacket`.
6. `IsAwayFromOrigin("nft", "channel-0", "nft/channel-0/nftClass")` → `false`.
7. `unprefixedClassID = "nftClass"`, `voucherClassID = "nftClass"`.
8. `TransferOwner(ctx, "nftClass", "token1", GetEscrowAddress("nft", "channel-X"), attacker_address)` → succeeds.
9. `"nftClass"/"token1"` is now owned by `attacker_address`. `userA`'s NFT is stolen. [10](#0-9)

```go
func (k Keeper) processReceivedPacket(ctx sdk.Context, packet channeltypes.Packet,
	data types.NonFungibleTokenPacketData,
) error {
	receiver, err := sdk.AccAddressFromBech32(data.Receiver)
	if err != nil {
		return err
	}

	isAwayFromOrigin := types.IsAwayFromOrigin(packet.GetSourcePort(), packet.GetSourceChannel(), data.ClassId)

	// create the escrow address for creating denom and minting nft
	escrowAddress := types.GetEscrowAddress(packet.GetDestPort(), packet.GetDestChannel())

	if isAwayFromOrigin {
		// since SendPacket did not prefix the classID, we must prefix classID here
		classPrefix := types.GetClassPrefix(packet.GetDestPort(), packet.GetDestChannel())
		// NOTE: sourcePrefix contains the trailing "/"
		prefixedClassID := classPrefix + data.ClassId

		// construct the class trace from the full raw classID
		classTrace := types.ParseClassTrace(prefixedClassID)
		if !k.HasClassTrace(ctx, classTrace.Hash()) {
			k.SetClassTrace(ctx, classTrace)
		}

		voucherClassID := classTrace.IBCClassID()

		if !k.nftKeeper.HasDenomID(ctx, voucherClassID) {
			if err := k.nftKeeper.IssueDenom(ctx, voucherClassID, voucherClassID, "", data.ClassUri, escrowAddress); err != nil {
				return err
			}
		}
		sdkCtx := sdk.UnwrapSDKContext(ctx)
		sdkCtx.EventManager().EmitEvent(
			sdk.NewEvent(
				types.EventTypeClassTrace,
				sdk.NewAttribute(types.AttributeKeyTraceHash, classTrace.Hash().String()),
				sdk.NewAttribute(types.AttributeKeyClassID, voucherClassID),
			),
		)

		for i, tokenID := range data.TokenIds {
			if err := k.nftKeeper.MintNFT(ctx, voucherClassID, tokenID, "", data.TokenUris[i], "", escrowAddress, receiver); err != nil {
				return err
			}
		}
	} else {
		// If the token moves in the direction of back to origin,
		// we need to unescrow the token and transfer it to the receiver

		// we should remove the prefix. For example:
		// p6/c6/p4/c4/p2/c2/nftClass -> p4/c4/p2/c2/nftClass
		unprefixedClassID := types.RemoveClassPrefix(packet.GetSourcePort(),
			packet.GetSourceChannel(), data.ClassId)

		voucherClassID := types.ParseClassTrace(unprefixedClassID).IBCClassID()
		for _, tokenID := range data.TokenIds {
			if err := k.nftKeeper.TransferOwner(ctx,
				voucherClassID, tokenID, escrowAddress, receiver); err != nil {
				return err
			}
		}
	}

	return nil
```

```go
func ParseClassTrace(rawClassID string) ClassTrace {
	classSplit := strings.Split(rawClassID, "/")

	if classSplit[0] == rawClassID {
		return ClassTrace{
			Path:        "",
			BaseClassId: rawClassID,
		}
	}

	return ClassTrace{
		Path:        strings.Join(classSplit[:len(classSplit)-1], "/"),
		BaseClassId: classSplit[len(classSplit)-1],
	}
}
```

```go
func (nftpd NonFungibleTokenPacketData) ValidateBasic() error {
	if strings.TrimSpace(nftpd.ClassId) == "" {
		return newsdkerrors.Wrap(ErrInvalidClassID, "classId cannot be blank")
	}

	if len(nftpd.TokenIds) == 0 {
		return newsdkerrors.Wrap(ErrInvalidTokenID, "tokenId cannot be blank")
	}

	if len(nftpd.TokenIds) != len(nftpd.TokenUris) {
		return newsdkerrors.Wrap(ErrInvalidPacket, "tokenIds and tokenUris lengths do not match")
	}

	if strings.TrimSpace(nftpd.Sender) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "sender address cannot be blank")
	}

	// decode the sender address
	if _, err := sdk.AccAddressFromBech32(nftpd.Sender); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid sender address")
	}

	if strings.TrimSpace(nftpd.Receiver) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "receiver address cannot be blank")
	}

	// decode the receiver address
	if _, err := sdk.AccAddressFromBech32(nftpd.Receiver); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid receiver address")
	}
	return nil
}
```
